8.bit CSS

This is just a dialog :)

8.bit CSS is a minimal CSS framework that uses no classes to give pages a retro look. You just need to include a single CSS file, and the page will immediately have an 80s style.

You can show a dialog by clicking here.

In this demo page, the dialog will automatically close after 3 seconds.
